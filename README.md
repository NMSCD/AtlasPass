### Requirements
- [NodeJS](https://nodejs.org/en/download/)

### Steps:
1. Clone this repository
2. In the directory where the `package.json` file is, run `npm i`
3. Run the webApp `npm run dev`
4. View it on [localhost:3000](http://localhost:3000)
